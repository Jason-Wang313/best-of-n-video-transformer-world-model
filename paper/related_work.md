# Related Work

This scaffold connects to work on model-based planning, video prediction, uncertainty estimation, and selected-tail evaluation. Its closest methodological neighbor is finite-pool top-score analysis, but the object being selected here is a rendered action-conditioned video future.

The work differs from token aliasing studies because the failure is inspectable in frame space. It differs from latent imagination studies because the selected video is paired with an executable action sequence and a rendered ground-truth counterfactual.
